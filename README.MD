## Tech Stack

- **Database**: MongoDB
- **File Storage**: Cloudinary
- **Authentication**: JWT

## Quick Start

1. Clone the repository
2. Run `npm run install-all`
3. Set up environment variables
4. Run `npm run dev`

## Environment Setup

Copy `env.example` to `.env` in both root and backend directories and configure your variables.
